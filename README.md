# Azure ADF Data Migration Project

## Project Overview
An end-to-end Azure Data Factory project that migrates data from SQL Server to Azure Data Lake Storage Gen2 using Full Load and Incremental Load pipelines.

## Technologies
- Azure Data Factory
- SQL Server
- Azure Data Lake Storage Gen2
- Self-Hosted Integration Runtime
- GitHub

## Features
- Full Load Pipeline
- Incremental Load using Watermark
- Parent-Child Pipeline Architecture
- Dynamic Datasets
- Parameterized Pipelines
- Error Handling# Azure ADF Data Migration Project
